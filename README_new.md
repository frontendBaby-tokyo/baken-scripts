# sanrentan cli

「sanrentan cli」は、競馬の3連単（1着→2着→3着の順番を当てる馬券）の組み合わせをランダムに生成するCLIツールです。

## インストール

```sh
npm install -g sanrentan-cli
```

## 使い方

コマンドラインで `srt` コマンドを実行します。

```sh
srt
```

実行すると「出馬数は？ >」と表示されるので、3以上の数字（出走馬の数）を入力してください。

例:

```
$ srt
出馬数は？ > 7
3連単: 2 → 5 → 1
```

## 仕様

- 入力した出馬数（3以上の整数）から、ランダムに3頭を選び、1着→2着→3着の順で表示します。
- 入力が3未満や数字でない場合はエラーを表示します。
- 毎回ランダムな組み合わせを生成します。

## コマンド

- `srt` : 対話形式で3連単を1組生成します。
